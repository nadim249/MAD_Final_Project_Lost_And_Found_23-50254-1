## Overview

**Lost And Found** addresses the inconvenience of tracking lost or found items in a community (such as a university campus). The app allows users to create posts for lost or found items, view recent listings, and connect with owners or finders directly through the app.

## Features

- **Post Lost Items:** Users can submit details and photos of lost items.
- **Post Found Items:** Users can inform others about found items and include photos/descriptions.
- **List & Search Items:** Browse and search through lost & found posts by keywords or categories.
- **Contact Feature:** Easily connect with the finder or owner through the app (chat).
- **Profile Management:** Users can manage their posted items and profile.

## Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/nadim249/MAD_Final_Project_Lost_And_Found_23-50254-1.git
    cd MAD_Final_Project_Lost_And_Found_23-50254-1
    ```

2. **Open the project in Android Studio** (or your chosen IDE for mobile development).

3. **Set up dependencies:**
   - Ensure you have the correct SDK installed.
   - Sync Gradle or any required packages.

4. **Run the app** on an emulator or physical device.

## Usage

- Sign up or log in to the app.
- Choose an option to post or search for items.
- Fill out all mandatory fields when creating a post.
- Browse recent posts or use the search feature.
- Use the contact option to communicate safely within the app.
